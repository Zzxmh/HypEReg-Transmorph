\section{Supplementary Experimental Details}
\subsection{Model and Environment Configuration}
This supplementary document summarizes implementation details required for reproducibility:
\begin{itemize}
\item Framework and package versions: Python 3.12.10; PyTorch 2.12.0.dev20260408+cu128; antspyx 0.6.3; SimpleITK 2.5.3.
\item Hardware configuration: NVIDIA RTX PRO 6000 Blackwell Workstation Edition (97,887 MiB VRAM), driver 591.86; peak-memory audits from per-case evaluation logs indicate a single-GPU run profile for deep models.
\item Checkpoint management follows a unified manifest-and-version protocol aligned with the benchmark settings.
\item Inference is conducted under a standardized evaluation configuration with harmonized metric definitions across models.
\item Isolated forward profiling is executed in a clean no-gradient setting without label-warp or metric kernels. Measured pure-forward runtime/peak-memory for HypEReg-TransMorph and TransMorph are 0.4316 s / 5.422 GB and 0.4306 s / 5.421 GB, respectively.
\end{itemize}
